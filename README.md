# 时之笛 · Web Clone

用 Three.js 致敬《塞尔达传说:时之笛》的网页小游戏。非官方粉丝作品,仅供学习交流。

所有模型为程序化低多边形构建,所有音效与音乐由 WebAudio 实时合成,无任何外部素材。

## 运行

```bash
npm install
npm run dev      # 开发模式,默认 http://localhost:5173
npm run build    # 打包到 dist/
```

## 玩法

- 海拉鲁平原自由探索,昼夜实时循环(一昼夜约 4 分钟)
- 夜晚骷髅兵会从地下爬出来;水塘有吐石头的章鱼怪,用盾反弹石头可以一击反杀
- 挥剑割草有几率掉卢比和红心;翻滚撞树也会掉东西
- 西边山丘上有个宝箱,里面是心之容器(生命上限 +1)
- 妖精纳薇会在路上给你提示

## 操作

| 按键 | 功能 |
| --- | --- |
| W A S D | 移动 |
| 空格 | 翻滚(撞树有惊喜) |
| J / 鼠标左键 | 挥剑(三连击) |
| K / 鼠标右键(按住) | 举盾格挡 / 反弹石头 |
| Tab | Z 注视锁定 / 解除 |
| E | 调查、开宝箱 |
| O | 取出 / 收起陶笛 |
| 方向键 + 空格 | (陶笛模式)吹奏音符 |
| Q / ← → | 旋转视角,滚轮缩放 |
| H | 操作说明 |

## 技术

- Vite + Three.js(MeshToonMaterial 卡通渲染、InstancedMesh 草地、程序化地形)
- WebAudio 合成:陶笛(正弦 + 颤音)、BGM(原创旋律调度器)、全部音效
- 程序化动画:跑步、翻滚、三连斩、持盾、吹笛全部由代码驱动
